Detailed Description

template<std::size_t Idx, class TP>
struct tp::at< Idx, TP >

Performs access to elements of a type_pack by index.

Template Parameters
Idxindex
TPtype_pack

Provides the member typedef type, which is the element of a type_pack with the index Idx. If an index is greater than size of a type_pack, compiler error will happen.
