Detailed Description

template<typename T, class TP>
struct tp::remove_all< T, TP >

Removes all elements of a specific type.

Template Parameters
Tremoving type

Provides the member typedef type that is a type_pack with all elements from the original type_pack except all elements of a type T.

If a pack is empty, the resulting type is empty_pack.
