Detailed Description

template<class...>
struct tp::concatenate<... >

Concatenates type_packs and just_types.

Provides the member typedef type which is a pack of elements of concatenation operands. If type of operands is just_type, then the resulting type is type_pack with two elements. Allows to concatenate a type_pack with just_type. If just_type is the first operand, its type will be added to front of a pack, otherwise if just_type is the second operand and type_pack is the first, type will be concatenated to back.

Operation acts recursively and can concatenate more, that two operands.

Note that concatenate requires type wrapped either in just_type or type_pack. Attempt to concatenate type itself will cause a compiler error.
