template<class T, class U>
struct tp::is_not_equal< T, U >

Checks if the two type_packs or just_types is not equal.

Template Parameters
Tleft operand
Uright operand

Operation compares just_type with just_type, type_pack with type_pack and type_pack with just_type. The last comparsion is only allowed to type_pack, which has only one type. Provides the member constant value that is equal to true if the types are not equal, otherwise false. is_not_equal inherits std::integral_constant members.
