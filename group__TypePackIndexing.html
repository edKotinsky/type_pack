Detailed Description

Operations that return indexes of types.

Indexing operations, which divides into following categories:

  • Operations that check if a type exists in a type_pack. These are: all_of, any_of, none_of, contains.
  • Operations that return an index of a type if this type exists in a type_pack. These are: find, find_if.
  • Operations that return count of specific types or types, on which a predicate returns true: count, count_if. Indexing operations provide a constant member value as a result.
