Detailed Description

template<template< typename... > class F, class TP>
struct tp::count_if< F, TP >

Returns the number of elements satisfying predicate condition.

Template Parameters
Funary predicate metafunction, that provides the member constant value with the type bool. The signature must be equivalent to the following: template <typename T> struct pred;
TPtype_pack

Provides the member constant value of type std::size_t. Counts elements on which predicate returns true. Inherits std::integral_constant members.

If a pack is empty, result will be 0.

public:

};
