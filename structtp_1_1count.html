Detailed Description

template<typename T, class TP>
struct tp::count< T, TP >

Returns the number of elements with the specific type.

Template Parameters
Ttype to look for
TPtype_pack

Provides the member constant value with the type std::size_t. Counts the elements that are equal to type T (using std::is_same). Inherits std::integral_constant members.

If a pack is empty, result will be 0.
