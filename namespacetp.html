Detailed Description

Main namespace of a library.

TypePack Library provides metaprogramming type list and a great set of operations with it like indexing access, generating, copying, comparsion, and even sorting.

In fact, it is a hobby project, caused by my wish to deeply explore C++ metaprogramming and other topics, like doxygen usage and customisation, documentation writing, testing, etc.
