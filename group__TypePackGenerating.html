Detailed Description

Operations that create a new type_pack.

Generation operations, what generate new type_pack from the other type_pack, or create new type_pack from the other data (e.g. generate creates a type_pack with N elements of a specific type). Some of these operations generate new pack from the other, conditionally, by index or by position. Generation operations provide member typedef type, which is a type_pack with some types (or empty_pack in some cases) as a result.
