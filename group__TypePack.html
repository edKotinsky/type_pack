Detailed Description

A C++11 type list implementation.

Type pack is a list of types, stored in a template parameter pack. Instead of a classic type list, which has head type and recursive tail, type pack has not. Because template parameter packs was presented in 11 C++ standard, type_pack requires C++11 or greater.
