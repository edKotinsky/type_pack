Detailed Description

Utilities, that are useful with the other type_pack operations.

Utilities allow to apply third party metafunctions with the type_pack operations.

E.g. part_caller allows to apply metafunctions, which takes two or more template parameters, in case when metafunction with one template parameter is needed - "partially specialize" it.
