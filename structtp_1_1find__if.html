Detailed Description

template<template< typename... > class F, class TP, std::size_t From = 0>
struct tp::find_if< F, TP, From >

First the first type in a type_pack satisfying predicate condition.

Template Parameters
Funary_predicate
TPtype_pack
Fromstart index (optional)

Provides the member constant value with the type std::size_t, which is an index of a type in a type_pack. If the type is not found, index will be equal to the size of a type_pack.

If From index is given, searching will started from it. By default this index is 0.
