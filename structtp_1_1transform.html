Detailed Description

template<template< typename... > class F, class TP>
struct tp::transform< F, TP >

Generates new type_pack from other type_pack by applying to its elements transforming metafunction.

Template Parameters
Funary transforming metafunction, that is take one template type and provide the member typedef type. The signature of transforming metafunction must be equivalent to the following: template <typename T> struct trans;.
TPtype_pack

Provides the member typedef type that is a type_pack, filled with types, generated by transforming metafunction.
